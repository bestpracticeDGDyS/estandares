Esta sección debe hablar de los principios a tener en cuenta cuando se crean o evolucionan productos digitales.

Algunos conceptos a tener en cuenta:

- Diseño de productos centrado en el usuario.
- Diseñar soluciones que tengan en cuenta los diferentes canales de contacto con el usuario.
- Los productos deben ser simples y usables.
- Los productos deben ser accesibles a todos los usuarios.
- Los productos deben entenderse como parte de una estrategia.

También se deben señalar ciertas soluciones generales como:

- **buenosaires.gob.ar** como sitio web institucional de gobierno.
- **Mi BA** como sistema de identificación para productos digitales que requieran un usuario.
- **Boti** como sistema de asistencia conversacional.
